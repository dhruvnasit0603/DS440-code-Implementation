# DS440-code-Implementation
## Requirements & Setup
### To run the Quadruped Robot Demo Dashboard without errors, ensure your system meets the following requirements.
1. Python 3.8 or higher
2. pip (Python package manager)
3. A good web browser (Chrome, Safari, Edge)
### The dashboard is built using Streamlit and standard Python libraries.
Install the required dependency: *pip install streamlit*
### Running the code.
1. Navigate to the project directory:*cd your_project_folder*
2. Run the Streamlit app: *streamlit run Quadruped_robot_demo_app.py*
